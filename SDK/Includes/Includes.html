Quesa Includes

The Quesa distribution provides all of the header files which define the Quesa API - this includes both the QuickDraw™ 3D API, and our extensions to it.

Applications which use the Quesa headers may not be binary-compatible with QuickDraw™ 3D 1.6 (since they may make use of extensions which are not present in Apple's QuickDraw™ 3D).
