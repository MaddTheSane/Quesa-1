Quesa Scripts

This directory contains various Perl scripts which were used in the development of Quesa.

These scripts are not required for developing with Quesa, but are included in the distribution in case they are useful in the future:

  • Skeleton - Generates stub code from QD3D headers. Was used to to generate the initial Quesa source tree.
  • Status - Generates the status table for the docs.
  • Headeriser - Generated the Quesa headers from the source. Was used to generate the initial Quesa headers.

To run these scripts, you will need a copy of Perl
