Quesa Includes

The Quesa distribution does not currently provide the header files which define the QuickDraw™ 3D API - these headers must be obtained from an alternative source, since the copyright on these files belongs to Apple.

For information on how to obtain these headers, please see the Building Quesa page.

The QuickDraw™ 3D headers should be placed into the "SDK/Includes/Apple/" folder within the Quesa distribution.

The "SDK/Includes/Quesa/" folder contains the public Quesa headers. Applications which use these headers may not be compatible with QuickDraw™ 3D 1.6, since they may contain extensions to the QuickDraw™ 3D API.
